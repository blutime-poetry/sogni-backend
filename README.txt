
ISTRUZIONI PER IL DEPLOY:

1. Crea un nuovo repository su GitHub, ad esempio: sogni-backend-pdf
2. Carica i 2 file: server.js e package.json
3. Su Render.com:
   - Crea un nuovo Web Service
   - Seleziona questo nuovo repository GitHub
   - Start command: node server.js
   - Porta: 10000 (Render la rileva automaticamente)
4. Fai deploy
5. Endpoint funzionante: POST /pdf
